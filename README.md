# ✈️ Redox Airlines Analysis Dashboard

An interactive Power BI dashboard analyzing flight operations, revenue performance, and route profitability for Redox Airlines — built to surface actionable insights for airline management and operations teams.

![Redox Airlines Dashboard](Redox_Airlines_Dash.JPG)

---

## 📊 Dashboard Overview

This dashboard provides a comprehensive view of airline performance across three key areas:

**Operations** — Total flight volume, route-level traffic, and monthly demand trends  
**Revenue** — Total revenue, average ticket price, fuel cost efficiency by aircraft type  
**Profitability** — Route-level profit and loss, revenue share by route ID

---

## 🔍 Key Insights

- **9,636 total flights** analyzed across all routes
- **$2.74M average ticket price**, with revenue performance varying significantly by route
- **ORD-LAX** is the most profitable route at ~$51M, while **ORD-SLC** trails at ~$2M
- **August and July** are peak travel months for the Fort Lauderdale route (1,025 and 1,013 flights respectively)
- **B737** has the highest fuel cost per seat mile at $2, compared to A319 ($1) and A320 ($0) — a key factor in route cost planning
- The **ORD-PHL route** recorded 315 flights, representing a measurable share of the Chicago hub's operations

## 💡 Features

- **Route Profitability Chart** — Quickly identifies highest and lowest earning routes across the network
- **Aircraft Fuel Efficiency Comparison** — Breaks down fuel cost per seat mile by aircraft type (A319, A320, B737)
- **Seasonal Demand Analysis** — Monthly flight volume trends for key routes (e.g., Fort Lauderdale)
- **Revenue Share by Route ID** — Proportional breakdown of total revenue contribution per route
- **KPI Cards** — At-a-glance metrics for total flights, total revenue, and average ticket price

---

## 🚀 How to Use

1. Clone or download this repository
2. Open `Redox_Airlines.pbix` in Power BI Desktop
3. Refresh the data connection if prompted
4. Use the interactive filters and slicers to explore routes, aircraft types, and time periods
